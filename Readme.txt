The idea came to me when I had to create notes or todos for sujects on my phone and had to label them with different subjects. This app provides the same but with better user interaction
It has data presistance using Async Storage and cross platform compatibility as it is built using React Native. Web version is still under build and after that you'll be able to access 
your companion on browser also.
